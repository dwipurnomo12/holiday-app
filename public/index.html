ENIGMAAPI

Kalender Nasional

API untuk cek hari libur nasional dan cuti bersama di Indonesia

Format Response

{
    "tanggal": "2023-01-1",
    "tanggal_display": "Minggu, 1 Januari 2023",
    "keterangan": "Tahun Baru 2023",
    "is_cuti": false
}

Cara Akses API

  • Data hari libur tahun sekarang /api
  • Data hari libur tahun 2020 /api?year=2020
  • Data hari libur bulan 5 tahun saat ini /api?month=5
  • Data hari libur bulan 5 tahun 2023 /api?month=5&year=2023

Cara Kerja API?

EnigmaAPI mengambil data dari tanggalan.com dengan scraping terjadwal setiap 1 bulan sekali via GitHub Actions.
